Web Design

Este é um parágrafo de exemplo criado para cumprir os requisitos da atividade proposta.
Ele possui configurações específicas de estilo, como cor de fundo cinza e texto em itálico.
O objetivo é demonstrar o uso de seletores CSS para modificar a aparência de elementos HTML.
O texto está centralizado e utiliza a fonte Tahoma com um espaçamento entre linhas duplo.
Esta é a quinta linha necessária para completar o requisito mínimo do exercício de Web Design.
